🌦️ Weather App - Real-time Weather Forecast
A modern web application that allows users to search and retrieve real-time weather information for cities worldwide with high accuracy.

🚀 Key Features
Search Functionality: Get instant weather updates by city name.
Detailed Insights: Displays temperature, humidity, wind speed, and sky conditions.
Responsive Design: User-friendly interface that works seamlessly across all devices (Mobile, Tablet, Desktop).
Live Data: Powered by the OpenWeatherMap API for up-to-the-minute accuracy.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+).
Framework: React.js.
Data Fetching: Axios / Fetch API.
External API: OpenWeatherMap API.

📦 Installation & Setup
1. Clone the repository:
git clone https://github.com/SyPhu243/Weather-App

2. Navigate to the project directory:
cd Weather-App/weather-app-react (for Frontend)
cd Weather-App/weather-app (for Backend)
4. Install dependencies:
npm install
5. Start the development server:
npm start
