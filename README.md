# 🌦️ Weather App - Ứng dụng xem thời tiết

Dự án này là một ứng dụng web cho phép người dùng tra cứu thông tin thời tiết tại các thành phố trên thế giới một cách nhanh chóng và chính xác.

## Tính năng nổi bật
* **Tìm kiếm:** Xem thời tiết theo tên thành phố.
* **Thông tin chi tiết:** Hiển thị nhiệt độ, độ ẩm, tốc độ gió và trạng thái bầu trời.
* **Giao diện:** Thân thiện, hỗ trợ phản hồi tốt trên nhiều thiết bị.
* **Dữ liệu thực:** Kết nối trực tiếp với API thời tiết (OpenWeatherMap).

## 🛠️ Công nghệ sử dụng
* **Frontend:** HTML5, CSS, JavaScript.
* **Framework:** React.
* **API:** OpenWeatherMap API.
